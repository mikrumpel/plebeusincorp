После «cd» обязательно добавьте пробел. Эта команда, которая представляет собой сокращение от «change directory» (изменить каталог), является основной командой для перехода в другой каталог.
Сделайте это после команды «cd». Убедитесь, что между «cd» и путем к каталогу есть пробел.
Например, команда может выглядеть так: cd Windows\System32 или cd D:.
По умолчанию местоположением всех папок считается жесткий диск (например, «C:»), поэтому букву жесткого диска вводить не нужно.

# Советы

Изменить каталог необходимо, чтобы отредактировать или удалить файл.
Некоторыми общими командами для работы с каталогами являются следующие:
D: или F: – изменить каталог на дисковод или подключенный флеш-накопитель.
cd .. – переместиться из текущего каталога на одну папку вверх (например, из «C:\Windows\System32» в «C:\Windows»).(пробел не обязателен)
cd /d – одновременно изменить диск и каталог. 
Например, если в командной строке вы находитесь в корневом каталоге диска «D:», введите «cd /d C:\Windows», чтобы перейти в папку «Windows» на диске «C:».
\ – перейти в корневой каталог (например, системного диска).

# Команды

  dir - Выводит список файлов и подкаталогов, находящихся в заданном каталоге.
  dir *.exe — выводится информация обо всех файлах с расширением EXE из текущего каталога;
  cd c:\catalog — установить текущим каталог с именем catalog на диске C;
  cd[ ].. — переход на каталог выше (пробелы необязательны);
  cd[ ]\ — переход в корневой каталог (пробелы необязательны);
  md books — в текущем каталоге создать подкаталог books;
  md c:\archives — в корневом каталоге на диске С: создается каталог archives;
  rd games — удаление каталога games в текущем каталоге.
  copy name.txt d:\t (копирует файл name.txt на диск d:\ в каталог t)

  copy name.txt+name1.txt newname.txt (объеденит файлы в новый файл)

  copy name.txt name1.txt (копирует содержимое name.txt d name1.txt, если файл name1.txt уже был будет запрос на копирование)

  copy name.txt d:\t\2.txt (копирует name.txt в каталогe t в файл 2txt)

  * заменяет символ или несколько в названии файла

 # Перенаправление ввода-вывода

  dir > file.txt (результат dir будет записан в file.txt)
  ver >> file.txt (добавляет результак ver в file.txt)

  tree (вывод файлового дерева в постраничном режиме)

  md или mkdir name (создание каталога)
  rd или rmdir name (удаление каталога)
  при пустом каталоге прописывать ничего не нужно
  rd/s - при использовании этого префикса, удалится папка, а также все вложенные в нее элементы. 
  Если Ваш каталог не пустой, и Вы не прописали это параметр, тогда у Вас ничего не выйдет.
  rd/Q - используется  при удалении без подтверждения.

  del name.txt (удаление файлов)

  echo (Вывод информации в любой источник назначения, файл, экран и т. д.)
  echo on
  echo off

fsutil file createnew filename.txt 0 (создание файлов)
Заменять fileSize фактический размер файла в байтах.

fsutil file createnew filename.txt fileSize

Например, чтобы создать файл размером 100 КБ, замените fileSize 1000. Тогда команда будет выглядеть так.

fsutil file createnew filename.txt 1000
